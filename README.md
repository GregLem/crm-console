# Консольная CRM "Учёт заявок" + Веб-версия на Django

Проект включает две реализации учёта заявок:
- **Консольная версия** на чистом Python с сохранением в CSV.
- **Веб-версия** на Django с канбан-доской, аналитикой и экспортом в Excel.

---

## Консольная версия

Простая консольная программа для учёта заявок клиентов. Данные сохраняются в CSV-файл, что позволяет открывать их в Excel или Google Sheets для дальнейшего анализа.

### Возможности

- **Добавление заявки** – имя, телефон, услуга, автоматическая дата/время.
- **Просмотр всех заявок** – красивая таблица в консоли с форматированием.
- **Поиск по телефону** – ищет по части номера (только цифры).
- **Удаление заявки** – по номеру из списка.
- **Редактирование заявки** – можно изменить имя, телефон или услугу.
- **Статусы заявок** – каждой заявке можно присвоить статус: `новая`, `в работе`, `выполнена`, `отменена`.
- **Фильтр по дате** – просмотр заявок, добавленных сегодня.
- **Автоматическое сохранение** – после каждого изменения данные перезаписываются в `leads.csv`.
- **Загрузка при старте** – все сохранённые заявки подгружаются из файла.

### Технологии

- Python 3.11+
- Встроенные модули: `csv`, `datetime`, `os`, `pathlib`
